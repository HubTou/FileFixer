------------ DISK EDIT HELP ------------
 PRESS [?] KEY FOR THE NEXT HELP SCREEN
----------------------------------------
SECTOR SELECTION:
 [LEFT ARROW]  PREVIOUS SECTOR.
 [RIGHT ARROW] NEXT SECTOR.
 [UP ARROW]    PREVIOUS TRACK.
 [DOWN ARROW]  NEXT TRACK.
 [G] GO TO A SPECIFIC SECTOR IN A FILE.
 [R] READ A SPECIFIC TRACK/SECTOR.
 [W] REWRITE A SPECIFIC TRACK/SECTOR.
 
SECTOR VIEW:
 [A] ASCII CHARACTERS VIEW MODE.
 [B] MODE ASCII-HEXA (HALF SCREEN EACH).
 [H] HEXADECIMAL BYTES VIEW MODE.
 [L] 6502 OP-CODES VIEW MODE.
 [X] HEXA MASK AND/OR/XOR SELECTION.

BYTE SELECTION:
 [I] UP A LINE.      [M] DOWN A LINE.
 [J] LEFT A BYTE.    [K] RIGHT A BYTE.
 [P] POSITION CURSOR AT A SPECIFIC BYTE.
 [U] GO TO THE TOP OF THE SCREEN.
 
------------ DISK EDIT HELP ------------
 PRESS [?] KEY FOR THE NEXT HELP SCREEN
----------------------------------------
EDITION:
 [0 TO 9] SELECT AN EDITION BUFFER.
 [ENTER]  ENTER EDITION MODE.
 [ESCAPE] LEAVE EDITION MODE.
 [Z] CANCEL ALL MODIFICATIONS.
 [-] RESTORE THE CURRENT BYTE.
 [<] REMOVE THE CURRENT BYTE.
 (=] FILL END OF BUFFER WITH BYTE.
 [>] ADD A BYTE (AT CURRENT POSITION).

GENERAL MODE:
 [C] COPY THE SECTOR ON THE OTHER DRIVE.
 [D] DIRECTORY COMMAND MODE.
 [E] ERASE/FORMAT TRACK/DISK.
 [F] FIND A BYTE/STRING PATTERN.
 [N] TOGGLE HEXA/DECIMAL NUMBERING.
 [O] INPUT/OUTPUT PARAMETERS SELECTION.
 [Q] QUIT THE DISKFIXER TO MONITOR.
 [S] SLOT AND DRIVE SPECIFICATION.
 [T] TYPEFACE MASK SELECTION (FLSH/INV).
 [V] VTOC EDITION MODE.
 
------------ DISK EDIT HELP ------------
GENERAL COMMANDS (CONTINUED):
 [Y]    TOGGLE ASCII FILTER.
 [.]    SPECIFY WINDOW ORIGIN.
 [?]    CALL FOR HELP. (DISPLAY THIS)
 [-->!] GO TO BEGINNING OF NEXT LINE.
 
KEYBOARD EQUIVALENTS:
 [CTRL-H] <===> [LEFT ARROW] KEY
 [CTRL-I] <===> [-->!] KEY
 [CTRL-J] <===> [DOWN ARROW] KEY
 [CTRL-K] <===> [UP ARROW] KEY
 [CTRL-M] <===> [RETURN] KEY
 [CTRL-Q] <===> [Q] KEY
 [CTRL-U] <===> [RIGHT ARROW] KEY

SPECIAL COMMANDS:
 [CTRL-C] TOGGLE LW CASE/NORMAL MODE.
 [CTRL-P] SCREEN HARDCOPY ON PRINTER.
 [CTRL-S] PRINTER SLOT NUMBER.
 [CTRL-T] TOGGLE MASK/UNMASK COMMAND.
